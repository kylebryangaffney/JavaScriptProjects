# JavaScriptProjects

## Overview
This repository contains JavaScript projects created for The Tech Academy JavaScript course. It serves as a central place to store, manage, and track my progress as I build out projects

## Purpose
The goal of this repository is to centrally organize all course projects, document progress, and make it easy to review or update past work over time.


## Highlights
* **[Calculator](https://github.com/kylebryangaffney/JavaScriptProjects/tree/main/AdvancedJavaScriptProjects/Calculator)** – A basic 4 function calculator focusing clean design.
* **[Pizza Project](https://github.com/kylebryangaffney/JavaScriptProjects/tree/main/AdvancedJavaScriptProjects/PizzaProject)** – An interactive ordering application built to demonstrate DOM manipulation.
* **[Tic-Tac-Toe](https://github.com/kylebryangaffney/JavaScriptProjects/tree/main/AdvancedJavaScriptProjects/TicTacToe)** – A classic game featuring line drawing animations.
* **[Todo App](https://github.com/kylebryangaffney/JavaScriptProjects/tree/main/AdvancedJavaScriptProjects/TodoApp)** – A basic task management application to track daily objectives.
